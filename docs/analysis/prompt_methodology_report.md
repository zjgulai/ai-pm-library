1. **角色定义** → 明确AI身份和专业领域

4. **示例引导** → 提供1-2个完整示例
5. **约束条件** → 明确边界和限制
6. **评估标准** → 定义好输出的质量标准

## 六、方法论总结：高频描述与模式

### 1. 最高频方法论关键词
- **框架/结构化** → 将复杂问题拆解为可执行的模块
- **数据驱动** → 用量化指标替代主观判断
- **自动化** → 减少重复劳动，聚焦高价值决策
- **A/B测试** → 实验验证替代假设争论
- **分层/分级** → 从简单到复杂渐进式推进

### 2. 跨角色通用方法论
1. **输入-处理-输出**三段式结构
2. **目标-指标-行动**闭环管理
3. **假设-实验-验证**科学方法
4. **问题-分析-方案**决策流程
5. **现状-目标-差距**对标分析

### 3. 领域专用方法论
- **数据科学**: CRISP-DM流程、特征工程→建模→评估
- **产品开发**: MVP→PMF→规模化、用户故事→PRD→验收
- **增长运营**: AARRR漏斗、增长飞轮、实验驱动
- **内容创作**: 钩子→价值→CTA、选题→创作→分发→变现
- **战略管理**: SWOT→战略→OKR→执行→复盘
